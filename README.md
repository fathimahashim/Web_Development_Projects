# 🎙️ Text To Speech Converter

A simple web-based Text to Speech Converter built using **HTML, CSS, and JavaScript**. This project allows users to type text and convert it into speech using the browser's built-in speech synthesis feature.

---



## 📌 Features

* Convert text into speech 🎤
* Multiple voice selection 🔊
* Clean and simple UI 🎨
* Responsive design (basic)
* Easy to use interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Speech API

---


---

## ⚙️ How It Works

1. User enters text in the textarea
2. JavaScript reads the text
3. Web Speech API converts text into speech
4. User can select different voices

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/fathimahashim/Web_Development_Projects.git
```

2. Open the project folder
3. Open `index.html` in your browser
4. Start typing and click **Listen** 🎧
